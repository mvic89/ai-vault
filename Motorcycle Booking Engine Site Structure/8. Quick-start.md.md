## What is .osm.pbf?

**PBF (Protobuf Binary Format)** = Compressed OpenStreetMap data

- âœ… **Correct format** - Yes, this is the standard format
- âœ… **Highly compressed** - 10-30x smaller than XML
- âœ… **Fast to process** - Binary format is efficient
- âœ… **Industry standard** - Used by all major OSM tools

**File sizes:**

- Luxembourg: 25 MB
- Germany: 3.8 GB
- France: 3.6 GB

## CSV Output Preview

Your CSV will have these columns:

**22 total columns** including:

- GPS coordinates

- Phone/email/website
- Opening hours
- Service tags
